# Bio-data Form

This repository contains a **personal Bio-data Form** created using **HTML, CSS, and JavaScript**. The form collects personal, educational, and contact information, and displays the submitted data dynamically.  

---

## **Project Structure**
---

## **Features**

- Responsive form layout for desktop and mobile.
- Pre-filled fields with example personal and educational data.
- Dynamic display of submitted data below the form.
- Clickable Matric Number linking to a personal webpage (dynamic based on the input).
- Clean and professional design using CSS.

---

## **How to Use**

1. Open `index.html` in a browser to view the form.
2. Fill in your details or modify the pre-filled data.
3. Click **Submit** to display the submitted bio-data below the form.
4. Click the **Matric Number link** to open the personal webpage (make sure to replace the URL in `script.js` if necessary).
